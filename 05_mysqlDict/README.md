### 项目功能
---

```
01 收集四六级，托福，雅思等词汇
02 基于mysql数据库和php脚本在web上显示词汇
03 基于python和php脚本的在线单词学习

```


### 项目结构
---

```
mysqlTest.php         # 用于测试能否连接到mysql数据库
mysqldict.php         # 脚本连接到mysql数据库，打印数据库中的单词数据，包含托福，雅思，GRE等数据

01_word-Python-PHP    # 是一个独立但相关的项目


word                  # 所有单词的txt文件，mysql数据库中的单词来源

```
